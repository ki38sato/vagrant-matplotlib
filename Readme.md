
# vagrant-matplotlib

## Setup

- pyenv
- virtualenv
- requests
- numpy
- pandas
- statsmodels
- matplotlib
- scipy

## Usage

- Edit Vagrantfile (ip, vm_name, synced_folder)
- Edit python version : ansible/site.yml#pyenv_python_version
- vagrant up
- vagrant ssh (or vagrant 192.168.200.70 default settings)
